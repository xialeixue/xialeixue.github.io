how to install hexo

  1. hexo init I meet a mistake

  2. when you want to write a blog

    • start typora
    • write a blog
    • save into ~/blog/source/_post/xxxxx.md
    • sudo hexo clean
    • sudo hexo g

    then you can see your blog

  3. we want our blog deploy in the github,so

    • cnpm install –save hexo-deployer-git
    • _config.yml file need to be motified
      • type: git
      • repo: https://github.com/xialeixue/xialeixue.github.io.git
      • branch: master
    • you may be told that “please tell me who you are “
      • nano home/kyrie/blog/.deploy_git/.git/config
      • [user] name=xialeixue email=xialeixue@hotmail.com

  4. every time we want to push new blog to my hexo blog

    • move .md file to _post directory
    • sudp hexo g
    • sudo hexo d

  5. if you want change theme,it’s easy for you

    • find the theme of others blog in the github
    • sudo git clone https://github.com/iissnan/hexo-theme-next.git themes/nextMist
    • change theme setting in _config.yml

  6. it is obviously that the most often use script are

    • sudo hexo clean
    • sudo hexo g
    • sudo hexo s(to see local effect)
    • sudo hexo d
